Approach

The work splits along two axes that mostly don't collide: a rename (mechanical, repo-wide) and a set of behavior edits (the new exploration phase + the aspect-gate fix + the learnings rule). The rename must land first (D1) so the behavior edits target the renamed paths (skills/logo/SKILL.md, skills/logo/scripts/svg-metrics.mjs). After that, the script change (svg-metrics.mjs + tests + code-metrics.md) and the SKILL.md rewrite (new Phase 3, phase renumber, learnings rule) edit disjoint files and can proceed in parallel. A final conformance task runs the four hygiene lints, the audit, skill-eval, and the selftest. Deterministic arithmetic stays in the script (Inv-3); the new gates preserve the non-interactive contract (Inv-6).
