The §H shape (INV-2) — read before writing any gate

Every hard condition is presence, ordering, or an integer comparison the script performs. Where genuine judgement is required, the model records a machine-comparable value, never a self-graded verdict — a "stamp consistent per dimension" token is satisfiable without reading a word, which is why D10 replaced it. Both disagreement gates use the same escape: mismatch → a required non-empty rationale, never a hard fail, so there is no incentive to rewrite the note to match the number.

Risks & mitigations

  • A gate wearing a presence costume. The failure mode the grill caught (D10). Any check satisfiable by stamping a token without reading is not a gate. T4's note-vs-score must be an integer comparison the script performs; T9 re-reads each gate and asks "can a model pass this without doing the work?"
  • Model arithmetic sneaking back in. The live bug. Every number — modal, weighted, renormalized, untested-%, band — is computed in T5 and only read by the model. T9 greps SKILL.md for any instruction to compute, average, or weight.
  • Blocking every run. Wiring the gate (T6) before it can pass, or making rationale-required the default rather than the mismatch case, turns a calibration aid into a hard stop on normal runs. T6 comes after T5, and both disagreement gates fire only on disagreement.
  • Regressing the citation gate. The one half of the contract that worked (INV-3). Frozen: check-citations.mjs is not modified by this story at all (its --stamp is 260721-z5n). T9 asserts an empty diff for it.
  • Bloating the panel's artifact. D9's collapsed <details> keeps the default reading experience (scores + notes) unchanged. T2 must default to collapsed and place the block under the notes, not above them.
  • Over-reaching into output (b). (b) has no scoring anchors; adding them is explicitly deferred (D11). T8 adds method only, and T9 asserts no new anchors in interviewer-notes-skeleton.html.
  • The headless override's residual risk. AUTO-PICK means a run can score against a bar no human saw. Not eliminable by design (maintainer's call) — mitigated only by data-rubric-provenance="synthesized-auto", so T7 must stamp it unconditionally, including on failure paths.
