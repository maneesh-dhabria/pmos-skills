Risks & mitigations

  • All-or-none silently becoming all-or-fail. The single most likely defect: a validator that requires data-level everywhere turns all 8 bundled archetypes red on this story's own merge and blocks /interview-guide outright. T4's rule is explicitly per dimension, and T5's "undescribed" fixture plus T6's corpus run are the two independent proofs.
  • Restating the contract in three places. §K violation waiting to happen — SKILL.md, the validator's error text, and output-shapes.md all want to describe the attribute. Only scorecard-skeleton.html defines it; the other three cite it.
  • Authoring the ceiling into level 3. The bar-inflation defect this epic exists to close can be baked straight into the descriptors. T2 pins the rule in the contract ("unprompted/up-front" is top-level only) so 260721-1a4 inherits it rather than re-deciding it.
  • Scope creep into the corpus. Backfilling "just a couple of archetypes while I'm here" would fold 1a4 back in and blow the story's skill-eval scope. T6 asserts an empty git diff for guidelines/.
