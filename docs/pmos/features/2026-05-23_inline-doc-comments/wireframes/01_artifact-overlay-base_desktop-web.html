§4.2 Resolver dispatch

The /comments resolve command reads each thread's sidecar entry, reads the originating skill from the artifact's <meta name="pmos:skill"> tag, and dispatches a fresh subagent per thread loaded with that skill's context.

§4.3 Apply-edit-at-anchor contract (FR-NEW-B)

Every originating skill exposes a uniform entrypoint: inputs {thread, artifact_path, anchor, body}, outputs {success, diff_ref, system_reply}. Idempotent.
