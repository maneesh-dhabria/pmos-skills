Overview

Build the /primer skill at plugins/pmos-learnkit/skills/primer/SKILL.md — the first skill of a new pmos-learnkit plugin. Five Claude Code phases produce a verified-source, audience-shaped HTML primer at {docs_path}/primer/{date}_{slug}.html with companion sections.json + sources.json sidecars. The plan derisks the substrate parameterization (FR-14 — touches every artifact) as the T1 tracer bullet, then scaffolds the new plugin's _shared/, then authors the skill in incremental phase-by-phase slices.

Done when: SKILL.md is ≤500 lines (target) / ≤800 (hard cap); bash plugins/pmos-toolkit/skills/feature-sdlc/tools/skill-eval-check.sh --target claude-code plugins/pmos-learnkit/skills/primer/ exits 0 with all [D] checks pass; bash plugins/pmos-toolkit/skills/feature-sdlc/tools/audit-recommended.sh plugins/pmos-learnkit/skills/primer/SKILL.md exits 0; rendering any pre-existing pmos-toolkit artifact post-substrate-change produces byte-stable user-visible attribution (toolbar + footer link text and href byte-identical, only the additive data-pmos-plugin="pmos-toolkit" attribute differs); all 4 reference files exist under reference/ with TOC in first 15 lines if >100 lines; SKILL.md body contains every section listed in spec §7.10 (verified by grep).
