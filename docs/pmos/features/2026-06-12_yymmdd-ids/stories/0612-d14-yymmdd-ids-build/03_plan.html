Plan — 0612-d14: year-prefixed backlog ids

Build story for epic 0612-jjs. Contract: ../../02_design.html. Single /execute run on feat/0612-d14; skill-eval'd before ship. Canonical task source: tasks.yaml (this page mirrors it).

Approach

Additive, no-migration extension of epic 0020's coordination-free id scheme. Change the canonical format + validator in its single home, then ripple the year-prefix into the minter, keep the uniqueness gate format-agnostic, extend the scheme to /mytasks (veto-able), and guard the change with a triple-accept regression test. The validator must keep all three arms — the central correctness trap is silently dropping the current <MMDD>-<rand3> arm.
