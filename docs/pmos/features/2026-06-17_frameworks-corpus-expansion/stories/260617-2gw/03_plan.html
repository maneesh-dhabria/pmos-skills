Approach

This is a deterministic data merge gated by an existing validator. The source candidates (learn-magazine/docs/pmos/frameworks/candidates/) hold 74 .md files, each with an embedded json record + a paired SVG. The plan: extract + splice the records via a script (never hand-edit the 1.1 MB file), copy the SVGs, then let validate-corpus.mjs be the gate, rebuild and eyeball the library. Pre-flight (epic design_doc #preflight) already proved the batch clean against this corpus, so the live build is largely a confirmation — the only real work is re-passing the diagram_anchors substring check and the merged distribution gate on this tree.
