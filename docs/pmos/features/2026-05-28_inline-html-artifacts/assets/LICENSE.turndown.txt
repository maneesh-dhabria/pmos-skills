Vendored third-party assets — license attribution
====================================================

The files in this directory include vendored builds of:

  1. turndown@7.2.4
       Source:  https://unpkg.com/turndown@7.2.4/dist/turndown.js
       Vendored as: turndown.umd.js

  2. turndown-plugin-gfm@1.0.2  (non-@joplin upstream)
       Source:  https://unpkg.com/turndown-plugin-gfm@1.0.2/dist/turndown-plugin-gfm.js
       Vendored as: turndown-plugin-gfm.umd.js

Both packages ship under the MIT License. The full license text for each
upstream is reproduced below. Copyright is retained by the original authors.

NOTE on plugin choice: spec FR-07 / Decision Log P3 referenced
@joplin/turndown-plugin-gfm@1.0.61 originally, but the @joplin fork dropped
the browser-friendly UMD bundle after that line — they ship CommonJS only.
The original (non-@joplin) `turndown-plugin-gfm@1.0.2` retains the
self-contained IIFE bundle (`dist/turndown-plugin-gfm.js`) we need for both
the browser viewer and the Node CLI shim. Same author (Dom Christie),
same MIT license, same GFM-table support.
