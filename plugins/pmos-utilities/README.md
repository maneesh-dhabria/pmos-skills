# pmos-utilities

**Help me maintain my environment.** pmos-utilities holds standalone diagnostics, cleanup, and meta-tooling that aren't part of a feature pipeline or a learning artifact — the skills you reach for to keep your machine and your skills healthy.

## Skills

- **/mac-health** — Diagnose a hot, slow, or battery-hungry Mac (orphaned processes, extension leaks, stale dev services, sleep blockers) before recommending cleanup.
- **/reflect** — Generate a paste-back retrospective for every pmos-skills skill used in the current session, severity-tagged and ready to hand to the skill author.

## Install

```
/plugin marketplace add maneesh-dhabria/pmos-skills
/plugin install pmos-utilities@pmos-skills
```
