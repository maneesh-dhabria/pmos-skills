# Creativity lenses — Tier 2 repertoire

The broader repertoire behind SKILL.md's Tier-1 lenses. Read when Phase 3 reaches the Tier-2 pass; give each lens a quick pass only — Tier 1 gets the deep thought. The 0-or-1-idea-per-journey rule applies here with extra force: most of these lenses will produce nothing for a given journey, and that's the expected outcome, not a failure.

**Floor, not ceiling:** this catalog (and Tier 1's) is a floor. If a stronger idea comes from outside the listed lenses, include it and name your own lens.

| Technique | Prompt |
|-----------|--------|
| **Shift the scale** | One axis, three probes: what if this was 10x the scope? What's the absolute smallest version that still delivers value? What if this was the ONLY feature — how would we make it perfect? |
| **Add constraints** | What if we restricted options to simplify the experience? |
| **Remove constraints** | What if we removed a limit that seemed necessary? (Inverse of the above and adjacent to Tier 1's friction pair — pick whichever fits; don't run all of them per journey.) |
| **Combine 2 unrelated things** | What if we merged this with something from a different domain? |
| **Solve multiple problems at once** | Can one change address several pain points simultaneously? |
| **Add friction** | Where would deliberate friction improve outcomes (e.g., confirmation, cooling-off)? |
| **Make them feel better & smarter** | How can we make the user feel competent, capable, and good about themselves? |
| **Build distribution within** | How can the product spread through its own use? |
| **Surprise** | Where can we exceed expectations in a delightful, unexpected way? |
| **Bundle** | What complementary experiences could we combine? |
| **Unbundle** | What should be separated into its own focused experience? |
| **Make it skeuomorphic** | What real-world analog could we mirror to make this instantly familiar? |
